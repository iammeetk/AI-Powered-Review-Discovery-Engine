# AI Review Discovery Engine — Requirements

## Objective

Build an AI-powered review analysis workflow for Spotify.

Goal:
Understand why users struggle to discover new music and repeatedly listen to familiar content.

---

## Data Sources

* Spotify Play Store reviews
* Spotify App Store reviews
* Reddit discussions
* Spotify Community

Rules:

* Last 12 months
* English only
* Max 500 records/source
* Remove duplicates

---

## Business Questions

1. Why do users struggle to discover new music?

2. What are the most common frustrations with recommendations?

3. What listening behaviors are users trying to achieve?

4. What causes users to repeatedly listen to the same content?

5. Which user segments experience different discovery challenges?

6. What unmet needs emerge consistently?

---

## Required Analysis

* Theme extraction
* Sentiment analysis
* User segmentation
* Intent detection
* Insight clustering

---

## Output

1. Data sources used

2. Key findings

3. Representative user feedback

4. User segments

5. Opportunities

6. One-slide workflow:
   Reviews → AI Analysis → Insights

Keep outputs PM-focused and presentation-ready.
